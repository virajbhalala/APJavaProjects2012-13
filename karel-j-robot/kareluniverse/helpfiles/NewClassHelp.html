You can now create your new class. Drop new methods and predicates into the class to create them. You can also drop Robot/Object type icons into the new class to create new helper objects.

Note that the new class also appears at the left-top in the robot/object class list.

You can send messages to "this" object within the new methods and predicates by dragging them from the "this" instance in the center part of the left pane.
